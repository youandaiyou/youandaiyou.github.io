What Pack 1 includes

Core materials

  • Implementation checklist (parents / teachers)
  • Classroom setup instructions (1 page)
  • Staff script and student rule: New chat = paste again
  • Printable quick reference sheet

Safeguarding framing

  • Scope and limits summary (plain language)
  • Supervision and duty-of-care reminder
  • “If uncertain, reload the declaration” rule
  • Link to the public Declaration Standard

Who Pack 1 is for

  • Parents who want a clear, repeatable safe start for AI use at home
  • Teachers who want boundaries set before classroom AI activities
  • Schools that want a baseline implementation bundle before formal adoption
  • Adults who choose the lowest-risk default interaction frame for themselves

When you need Pack 2 or Pack 3

Pack 1 provides baseline implementation guidance. If you require licensed, regulator-ready wording, declarations, and staff instructions aligned to this standard, use Pack 2 or Pack 3.
