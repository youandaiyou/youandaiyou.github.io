Purpose of This Page

This page provides the authoritative status of the School Implementation Standard, including current version, release date, and alignment to the public baseline. Institutions adopt the PDF version of this standard for internal use. This page exists solely to confirm whether an adopted version remains current or has been superseded.

Update & Reliance Notice

Updates to this standard are published on this page. Institutions are responsible for monitoring version status, determining whether updates affect reliance, and re-adopting the standard where required. No individual notifications are issued.

The Public Baseline Declaration establishes minimum child-safe conditions for AI use. The School Implementation Standard operationalises that baseline for institutional adoption and reliance.
