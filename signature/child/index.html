Load Child Mode before any AI use

A safer student context. Clear. Calm. Repeatable.

Step 1: Download the Child Mode image

Download Child Mode Image Then show/upload it to your AI before your first question.

Step 2: Start a new AI chat and upload the image (or show it to the AI if camera input is supported).
Step 3: Look for acknowledgement. If you start a new chat, reload the image again.

How to install Child Mode (2 minutes)

  1. Download the Child Mode image to the device.
  2. Open your AI tool and start a new conversation.
  3. Load the mode: upload the image or show it to the AI.
  4. Check confirmation: the AI should acknowledge it is responding in a student-safe way.
  5. Repeat: every new chat needs the image loaded again.

What this changes

  • Sets a clear student context before the first question
  • Promotes calm, respectful, age-appropriate language
  • Reduces adult framing and unsafe conversational drift
  • Encourages healthy questioning and “ask before assuming” behavior
  • Creates a repeatable routine: New chat = reload

For parents

Child Mode does not replace supervision. It sets the conversation stance so the AI responds appropriately to a child or student context from the first message.

For teachers and schools

Use Child Mode before classroom AI activities to keep boundaries clear and language appropriate.

  • Use before research, explanations, practice questions, and brainstorming
  • Supports guided learning, not shortcut answers
  • Simple classroom rule: New chat = reload Child Mode

Teacher script:
“Before we start, load Child Mode so the AI knows this is a student context. When it acknowledges the mode, you may ask your first question.”
