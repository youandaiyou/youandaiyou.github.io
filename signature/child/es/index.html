Modo Niño — Estándar de Declaración

Declaración pública y referenciable para interacción con IA.

Estándar universal de interacción (base)

Modo Niño define el estándar universal de interacción para el uso seguro de IA.

Es obligatorio cuando se utiliza IA con niños y puede ser adoptado voluntariamente por adultos que deseen operar dentro de las mismas restricciones predeterminadas de menor riesgo.

Esto no cambia la definición ni el propósito de Modo Niño. Lo establece como el marco predeterminado de menor riesgo.

Importante

Modo Niño no monitoriza, filtra ni supervisa el uso. La responsabilidad y supervisión adulta aplican en todo momento.

AI Frame™ siempre está activo.

Estado

Modo Niño no es un producto y no se vende.

Se ofrece gratuitamente como estándar público de declaración para que exista una base clara e inspeccionable antes de iniciar la interacción con IA.
